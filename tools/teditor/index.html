手順書画像作成君 - 使用方法

  • 画像をファイルメニューから開くか、Ctrl + V で貼り付けてください。
  • キーボードの数字キーで描画ツールを選択し、キャンバス上でドラッグして描画します。
    1. 矩形: 矩形を描写します
    2. 直線: 直線を描写します
    3. 矢印: 矢印を描写します
    4. ぼかし: 選択範囲にぼかし効果を適用します
  • Shiftキーを押しながら直線や矢印を描画すると、水平もしくは垂直方向に移動が制限されます。
  • 傍記対象にチェックマークをつけると描写した際に「①②」のような番号を自動的に付与します。
  • 傍記の位置はドラッグして移動できます。
  • フォント設定では傍記表示のフォントや大きさを調整できます
  • Ctrl + Z で戻る、Ctrl + Yでやり直し機能が使用できます。
  • 完成した画像は Ctrl + Sで保存、Ctrl + C でクリップボードにコピーできます。
